Quando for solicitado os testes na API:

- Iniciar o docker e dar play na imagem SQL_HAS - Nesta aula usaremos apenas o Banco Local
- Apontar a classe program.cs mudando trecho de "ConexaoSomee" para "ConexaoLocal"
- Rodar no terminal do projeto "dotnet ef database update" para criar o banco
- Copiar da pasta Aula14 a nova versão do script de tabelas gerais (script03_TabelasGerais) para seu projeto. Inclusive rodar este script no SQL server após ter executado o comando "dotnet ef database update".
